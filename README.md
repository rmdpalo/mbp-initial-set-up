# mbp-initial-set-up
A repo to help anyone set up a dev environment on a new macbook

## Apps 
```
$brew install —cask
	slack
	postman
	visual-studio-code
	zoom
	pgadmin4
```

Check brew.sh if there are any issues, command could’ve been changed

## Tech

	postgres
$brew services start postgresql
